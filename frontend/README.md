```bash

```

## Technologies

- React 18
- GSAP (GreenSock Animation Platform)
- Tailwind CSS
- Axios
- React Router

## Components

- **TextType**: Typing animation component with customizable speed and effects
- **Navbar**: Animated navigation with typing header
- **Compare**: Main comparison interface with glass morphism cards
